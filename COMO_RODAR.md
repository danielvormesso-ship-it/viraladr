# 🚀 Como rodar o ViraLAdr localmente

## Pré-requisitos
- Node.js 18+ instalado (https://nodejs.org)

## Passos

1. Extraia o ZIP numa pasta
2. Abra o terminal dentro da pasta
3. Execute:

```bash
npm install
npm run dev
```

4. Acesse: http://localhost:8080

Pronto! O Supabase e Railway já estão configurados.
